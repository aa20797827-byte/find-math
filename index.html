  1. 위 버튼으로 HTML 파일 다운로드 현재 대시보드 전체가 하나의 HTML 파일로 저장됩니다

  4. 생성된 URL을 카카오톡/SNS로 공유 링크를 복사해서 누구에게나 전송하세요 — 계정 없이 바로 열립니다
